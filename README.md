# React lots

⚠️ This is a fictional, fan-made project inspired by *The Sims 4*.

**Reactlots** is a real estate web application where users can browse and explore properties (lots) from *The Sims 4* as if they were part of the game world.

**Author:** 🦖 [Laira Almas](https://github.com/lairaalmas)

## Architecture

This project is part of a multi-repo setup:

- [reactlots-app](https://github.com/lairaalmas/reactlots-app) → Frontend + BFF layer
- [reactlots-api](https://github.com/lairaalmas/reactlots-api) → Static data API (WIP)

## Tech Stack

- React
- TypeScript
- Vite
- Bootstrap
- Node.js
